## 1. Description
Update project.
Module compare two directories and copy newest files from source to
destination directory.

## 2. Usage
```bash
update dir1 dir2 [-h] [-excl_dir dir_list]
```

### 2.1 Example
```bash
update.py dir1 dir2 -excl_dir dir3,dir4
```
